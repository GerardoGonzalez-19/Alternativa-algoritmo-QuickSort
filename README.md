Realiza un trabajo sobre el algoritmo de ordenamiento QuickSort en Java, investigando una alternativa donde el pivote se seleccione en la posición media del arreglo en lugar de usar el pivote al final.

El trabajo debe incluir:

Una breve explicación de cómo funciona QuickSort.

El código Java del QuickSort con pivote al final (versión original).

El código Java del QuickSort con pivote en la posición media.

Un programa de prueba que mida los tiempos de ejecución de ambos algoritmos con arreglos de diferentes tamaños.

Una tabla de resultados y gráficas comparativas.

Una conclusión explicando las diferencias de rendimiento entre ambas versiones.
